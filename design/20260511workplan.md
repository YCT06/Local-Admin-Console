## 技術決策紀錄

| 項目 | 決定 |
|---|---|
| CSS 策略 | tokens.css 為唯一 token 來源；Chakra UI 用 `var(--token-name)` 引用，不建 custom theme |
| Chakra UI 使用範圍 | 僅 Modal（UsersPage 新增使用者 dialog，提供 focus trap / ESC 無障礙） |
| 按鈕/Badge/Pill/表格 | 直接用 CSS class（`.btn`, `.pill`, `.table` 等），pixel-perfect |
| Locale 管理 | i18next 自身管理（含 localStorage），不放 Zustand |
| Zustand scope | `authStore`（isLoggedIn, user, token）、`uiStore`（theme, density） |
| Mock 資料 | API 函式回傳 Promise.resolve(MOCK_*)，切換真實 API 只改一行 TODO 註解 |
| Highcharts | 僅 StatusPage 趨勢折線圖（3 條線：CPU / 記憶體 / GPU） |
| Gauge 圓環 | 純 SVG stroke-dasharray，不用 Highcharts |
| MiniSpark | 純 SVG，不用 Highcharts |
