## 背景說明

初始建構（2026-05-11）以 Highcharts 實作 StatusPage 的 30 分鐘趨勢折線圖（CPU / 記憶體 / GPU 三條線）。  
Gauge 圓環與 MiniSpark 均為純 SVG，不受影響。

替換動機：
1. Highcharts 商業授權限制，ECharts（Apache 2.0）更適合本專案
2. 現有的 CJS/ESM interop workaround（`(module as unknown as { default }).default ?? module`）可完全消除
3. ECharts TypeScript 型別支援更完整（原生 `EChartsOption`，不需繞過型別）

---

## 技術決策

| 項目 | 決定 |
| ---- | ---- |
| ECharts 引入方式 | 方式 A：`import ReactECharts from 'echarts-for-react'`（完整引入） |
| 理由 | 專案僅一張圖，tree-shaken 帶來的 bundle 收益低，程式碼複雜度不划算 |
| vendor-charts chunk | 保留原 chunk 名稱，僅替換 `id.includes('highcharts')` → `id.includes('echarts')` |
| `vendor-charts` 大小 | 1,137 KB（gzip 373 KB），與 Highcharts 全量相近，chunk 分割策略不變 |

#### Highcharts → ECharts 設定對照

| Highcharts | ECharts |
| ---------- | ------- |
| `chart.backgroundColor: "transparent"` | 預設透明，無需設定 |
| `chart.margin: [12, 10, 28, 32]` | `grid: { top: 12, right: 10, bottom: 28, left: 32 }` |
| `chart.animation: false` | `animation: false` |
| `xAxis: { visible: false }` | `xAxis: { type: "category", show: false }` |
| `yAxis.gridLineColor: "#F0EFEC"` | `yAxis.splitLine.lineStyle.color: "#F0EFEC"` |
| `yAxis.labels.style` | `yAxis.axisLabel: { fontSize, color }` |
| `tooltip: { shared: true, valueSuffix: "%" }` | `tooltip: { trigger: "axis", valueFormatter: (v) => ... + "%" }` |
| `plotOptions.line.marker: { enabled: false }` | `series[].symbol: "none"` |
| `plotOptions.line.lineWidth: 1.5` | `series[].lineStyle.width: 1.5` |
| `series[].color` | `series[].lineStyle.color` + `series[].itemStyle.color` |
| `credits: { enabled: false }` | 預設無浮水印，無需設定 |
